"Jadi Pinter Tanpa Ribet"

Sanpustaka adalah aplikasi perpustakaan digital berbasis web modern yang memungkinkan pengguna untuk mencari, menjelajahi, dan membaca pratinjau buku dari seluruh dunia secara instan. Dibangun dengan antarmuka yang bersih, responsif, dan ramah pengguna.

Pencarian Cerdas: Terintegrasi langsung dengan Google Books API untuk hasil pencarian yang akurat dan luas.
Mode Gelap (Dark Mode): Lindungi mata Anda saat membaca di malam hari dengan satu klik.
Reader Mode (Mode Baca): Tampilan simulasi E-Reader yang nyaman dengan pengaturan ukuran font dan tema (Light, Sepia, Dark).
Desain Responsif: Tampilan optimal di Smartphone, Tablet, dan Laptop.
Detail Lengkap: Sinopsis, info penerbit, penulis, dan rating buku.
Animasi Halus: Pengalaman pengguna yang interaktif dan tidak membosankan.


ğŸ“– Panduan Untuk Pengguna (User Guide)

Cara Menggunakan:
Cari Buku: Ketik judul buku, nama penulis, atau topik (contoh: "Harry Potter", "Belajar Python") di kolom pencarian utama, lalu tekan Enter atau klik tombol "Cari".
Kategori Cepat: Klik tombol kategori di bawah kolom pencarian (seperti "Novel", "Teknologi") untuk rekomendasi instan.
Lihat Detail: Klik pada kartu buku mana saja untuk melihat sinopsis lengkap dan informasi detailnya.

Baca Buku:
Klik tombol "Baca Sekarang" untuk masuk ke Reader Mode (tampilan nyaman ala Kindle).
Klik "Google Play" untuk melihat versi lengkap atau membeli buku di toko resmi.
Ubah Tema: Klik ikon Bulan/Matahari di pojok kanan atas untuk mengganti tampilan antara Mode Terang dan Mode Gelap.

Tips:

Gunakan Reader Mode untuk membaca sinopsis panjang agar mata tidak cepat lelah.
Anda bisa mengubah warna latar menjadi "Sepia" agar lebih mirip kertas buku lama.
